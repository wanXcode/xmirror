# XMirror 版本说明

## 当前版本：v1.4.0

### v1.4.0 (2026-03-03)

#### 新增
- **内页翻译（v1）**：支持在存档内页一键翻译为中文（SiliconFlow/OpenAI 兼容接口）
- **译文缓存落库**：新增 `translations` 表，按 `post_id + target_lang + source_hash` 缓存译文
- **翻译 API**：新增 `GET /api/translate/:id?targetLang=zh-CN`

#### 优化
- **翻译交互简化**：由“原文/译文 Tab”优化为“单按钮切换”
  - 首次点击：翻译并展示译文
  - 再次点击：原文/译文一键切换
- **老页面升级**：支持批量重渲染历史存档页面，统一接入翻译按钮
- **部署方式**：补充 systemd 持久化启动方案（开机自启 + 环境变量）

#### 修复
- 修复 SiliconFlow 返回 JSON 结构不稳定导致的解析失败问题
- 修复服务切换阶段可能出现的 SQLite 只读写入异常（启动方式稳定后恢复正常）

### v1.0.0 (2026-02-25)

#### 功能特性
- **X/Twitter 内容存档**：输入 X 链接，自动抓取推文内容并生成可访问的镜像页面
- **支持普通推文**：处理带图片、视频的普通推文

- **推文标题显示**：长文章显示原文标题，普通推文显示内容前 50 字

## 技术实现

- **前端**：原生 HTML + CSS + JavaScript，无框架依赖
- **后端**：Node.js + Express
- **数据库**：SQLite 存储存档元数据（新增 translations 缓存表）
- **进程管理**：systemd 持久化启动（支持开机自启）
- **API 数据源**：FxTwitter API + SiliconFlow 翻译 API

## 文件结构

```
xmirror/
├── server.js          # 主服务程序
├── public/            # 前端静态文件
│   ├── index.html     # 首页
│   ├── style.css      # 样式文件
│   ├── app.js         # 主逻辑
│   └── i18n.js        # 国际化
├── archives/          # 生成的 HTML 存档文件
├── data/
│   ├── db.sqlite      # SQLite 数据库
│   ├── images/        # 下载的图片文件
│   └── videos/        # 下载的视频文件
├── VERSION.md         # 版本说明
└── README.md          # 项目说明
```
