# XMirror 版本说明

## 当前版本：v1.2.0

### v1.2.0 (2026-02-28)

#### 新增
- 国际化支持（i18n），自动检测浏览器语言（中文/英文）
- 语言切换按钮，支持手动切换中英文
- SEO 优化：添加 meta 标签、Open Graph、Twitter Card、结构化数据
- 分离 CSS 和 JS 文件，提高代码可维护性

#### 优化
- 首页标题和描述根据语言自动切换
- 页面加载性能优化（预连接、DNS 预解析）

---

### v1.1.0 (2026-02-26)

#### 新增功能
- **钉钉卡片优化**：添加 Open Graph 和 Twitter Card meta 标签，支持钉钉/微信/推特等平台的链接卡片预览
- **视频下载功能**：自动下载推文中的视频到本地服务器
- **视频播放支持**：HTML 存档页面支持视频播放

#### 优化
- 修复 UTF-8 编码问题，中文显示正常
- 优化图片在正文中的显示位置
- 添加隐藏删除功能（三连击底部区域）

---

### v1.0.0 (2026-02-25)

#### 功能特性
- **X/Twitter 内容存档**：输入 X 链接，自动抓取推文内容并生成可访问的镜像页面
- **支持普通推文**：处理带图片、视频的普通推文
- **支持长文章（Article）**：完整保存长文格式，包括标题、段落、列表等
- **图片本地化**：自动下载推文中的图片到本地服务器，避免外链失效
- **UTF-8 编码支持**：正确处理中文及特殊字符，无乱码

#### 页面功能
- **首页存档列表**：显示最近 10 条存档记录
- **推文标题显示**：长文章显示原文标题，普通推文显示内容前 50 字

---

## 技术实现

- **前端**：原生 HTML + CSS + JavaScript，无框架依赖
- **后端**：Node.js + Express
- **数据库**：SQLite 存储存档元数据
- **进程管理**：PM2 守护进程，开机自启
- **API 数据源**：FxTwitter API (api.fxtwitter.com)

## 文件结构

```
xmirror/
├── server.js          # 主服务程序
├── public/            # 前端静态文件
│   ├── index.html     # 首页
│   ├── style.css      # 样式文件
│   ├── app.js         # 主逻辑
│   └── i18n.js        # 国际化
├── archives/          # 生成的 HTML 存档文件
├── data/
│   ├── db.sqlite      # SQLite 数据库
│   ├── images/        # 下载的图片文件
│   └── videos/        # 下载的视频文件
├── VERSION.md         # 版本说明
└── README.md          # 项目说明
```
